In Ruby, classes organize and produce objects with similar methods and attributes.

Class Syntax:
class ClassName
#do something
end

Class names have to start with an uppercase leter and use CamelCase instead of snake_case. In the example above, ClassName is now a class.

Within your classes, you will want to create a method called initialize. Initialize takes as many arguments as you want.

You then use the @ operator before a variable to show that it is an instance variable, meaning it is attached to the instance of the class.

class HarryPotterCharacters
def initialize(first_name,last_name)
@first_name = first_name
@last_name = last_name
end
end

Names = HarryPotterCharacters.new("Ron","Weasley")

In this example, I created an instance Names of the class HarryPotterCharacters. Names has its own first_name of Ron and last_name of Weasley. These variables only belong to this instance = these variables are instance variables.
